Landing Page – Proyecto UX/UI

Landing page desarrollada a partir de un caso real, tomada como práctica para fortalecer mis habilidades en diseño UX/UI y su desarrollo en código.

El foco del proyecto estuvo en trasladar el diseño a una implementación en HTML y CSS, utilizando responsive design y buenas prácticas de maquetación.

Objetivo

- Practicar el traspaso de diseño a desarrollo.
- Aplicar mobile-first.
- Construir un layout responsive (mobile, tablet y desktop).
- Implementar microinteracciones y animaciones.
- Organizar el CSS con metodología estructurada (BEM).

Tecnologías utilizadas

- HTML5
- CSS3
- Flexbox
- CSS Grid
- Media Queries
- Animaciones on scroll con JavaScript


Enfoque UX/UI - Proceso de trabajo:

- Jerarquía tipográfica
- Consistencia en espaciados
- Sistema de colores con variables CSS
- Estados interactivos en botones y cards
- Microinteracciones (hover y reveal on scroll)

Estado del proyecto

EN curso: como práctica de diseño y desarrollo front-end.
